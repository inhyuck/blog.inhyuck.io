## 시험 전 정리 / 각종 질문답변

- Merge Sort 
  1. 평균 시간복잡도 O(n log n)
  2. 최악의 경우 시간복잡도 O(n log n) : 최악이 없다.

- Quick Sort 
  1. 평균 시간복잡도 O(n log n)
  2. 최악의 경우 시간복잡도O(n2)

- Heap Sort 
  1. Build Heap O(n log n)
  2. Heapify O(log n)
  3. Heap Sort O(n log n)

- Selection Algorithm
  1. 평균시간복잡도 O(n)
  2. 최악의경우일때는 언제이고, 그때의 시간복잡도는?
    => 이미 정렬된 경우가 최악이며, O(n2)
  3. Linear Selection Algorithm 시간복잡도 O(n)

- BST add / search / remove
  1. 평균 시간복잡도 O(log n)
  2. 최악의경우일때는 언제이고, 그때의 시간복잡도는?     
     => 편향트리인 경우, O(n)
